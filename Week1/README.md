Lab: Week 1 – ROS2 Workspace and First Node

## Lab Objective
The objective of this lab was to learn the basics of ROS2, create a ROS2 workspace, develop a Python package, and run a simple ROS node.

## Steps Performed

1. Verified ROS2 installation using terminal commands.
2. Created a ROS2 workspace named `ros2_ws`.
3. Built the workspace using `colcon build`.
4. Created a Python package called `my_first_pkg`.
5. Implemented a simple ROS node named `simple_node`.
6. Registered the node in `setup.py`.
7. Built the package and executed the node using `ros2 run`.
